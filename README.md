# Quick start react

![Tictactoe](./img/tictactoe.gif)
> Quick start **tic tac toe** of React

## Como utilizar
```sh
$ git clone https://github.com/TacioAntonio/tic-tac-toe.git
$ cd tic-tac-toe
$ npm i
$ npm start
```
